# 二次創作ガイドラインテンプレート

このファイルは、`character.json` の `fanworkGuidelines` に入れるためのテンプレート集です。

重要ルール:

- ユーザーが二次創作ガイドラインの追加を希望した場合だけ使う。
- キャラクター固有の禁止事項、年齢、外見、世界観、権利関係は、ユーザーが提供した範囲だけ反映する。
- 足りない禁止事項や許可範囲を推測で足さない。
- 迷う場合は `fanworkGuidelines` に公式化せず、`content/inbox/{characterId}/` に確認待ちとして残す。
- `{キャラクター名}`、`{連絡先文言}`、`{固有の注意事項}` はキャラクターごとに置き換える。

## 選び方

- 緩い版: 二次創作やAI生成を広く歓迎し、個人・同人・広告収益を許可寄りにする場合。
- 厳しい版: キャラクター保護、年齢・尊厳・禁止表現、商用利用制限を強めたい場合。

    "contact": "{連絡先文言。例: 商用利用、法人利用、大規模企画、判断に迷う利用は事前に相談してください。}",

          "公式作品・公式発表と誤認される表示は避けてください。"

      },
      {
        "title": "許可していること",
        "items": [
          "二次創作、AI生成、ファンアート、動画、配信、記事化を許可します。",

          "投稿先の仕様による広告収入や収益化は許可します。",
          "個人・同人規模の頒布は、常識的な範囲で許可します。"
        ]

      {
        "title": "相談が必要なこと",
        "items": [
          "商用利用、法人利用、継続的な商品展開、大規模な販売・広告利用は事前相談とします。",
          "公式企画、公式素材、公式発表と誤認されやすい利用は事前相談とします。",
          "公式ロゴ、公式画像、配布素材を大きく使う商品化は事前相談とします。"

          "公式素材、ロゴ、資料をそのまま再配布・販売する行為は禁止します。"

          "{固有の注意事項があればここに入れる。なければこの項目は削除してよい。}"

    ]

## 運用メモ

- 緩い版でも、公式誤認、第三者侵害、公式素材の無断再配布は禁止として残す。
- 厳しい版でも、ユーザーが明示していないキャラクター固有の禁止事項は追加しない。
- 商用利用の扱いは、`許可`、`応相談`、`禁止` のどれかをユーザーに確認する。
- `status` は、ユーザーが公開ルールとして確定した場合は `official`、確認待ちなら `draft` または inbox メモにする。
