# 受け渡しメモ

残念院ちる子のデスクトップアクセサリ紹介サイトです。
HTML/CSSだけで動く静的ページなので、GitHub Pagesなどにそのまま配置できます。

## 入っているもの

- `index.html`: 紹介サイト本体
- `styles.css`: サイトの見た目
- `assets/desktop-chiruko-booth.png`: BOOTH掲載画像
- `assets/chiruko-startup.png`: 起動時の画面
- `assets/chiruko-game.png`: ゲームで遊んでいる画面
- `assets/chiruko-music.png`: 音楽を再生している画面
- `assets/chiruko-stand-reading.png`: 透過済みミニキャラ立ち絵（読書）
- `assets/chiruko-stand-running.png`: 透過済みミニキャラ立ち絵（移動）
- `assets/chiruko-stand-sleeping.png`: 透過済みミニキャラ立ち絵（おやすみ）
- `assets/chiruko-stand-sitting.png`: 透過済みミニキャラ立ち絵（待機）
- `booth-copy.md`: BOOTHや紹介欄に使える文章案
- `README.md`: 使い方と構成メモ

## デザインの方向性

デモ用ではなく、本番公開を想定したシンプルなLPです。
テーマは「余白」「パステルカラー」「キュートな丸ゴシック系フォント」です。
見出しは漢字対応の `M PLUS Rounded 1c` を優先して指定しています。
CharactorCameoPJの黒金シックな雰囲気には引きずられすぎず、ちる子の衣装やアプリ画面が映える、明るくかわいい見せ方にしています。

## 確認方法

`index.html` をブラウザで開くと確認できます。
ファイル一式をそのまま公開先へ置けば表示できます。
