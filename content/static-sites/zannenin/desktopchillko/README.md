# デスクトップちる子 紹介サイト

残念院ちる子のデスクトップアクセサリを紹介するための、HTML/CSSだけで動く静的サイトです。
ナビゲーションや仮の公開情報を置かず、パステルカラー、余白、丸ゴシック系の文字で小さくかわいい本番向けLPとして作っています。

## ファイル構成

- `index.html`: 紹介ページ本体
- `styles.css`: デザイン設定
- `assets/desktop-chiruko-booth.png`: BOOTH掲載画像
- `assets/chiruko-startup.png`: 起動時の画面
- `assets/chiruko-game.png`: ゲームで遊んでいる画面
- `assets/chiruko-music.png`: 音楽を再生している画面
- `assets/chiruko-stand-reading.png`: 透過済みミニキャラ立ち絵（読書）
- `assets/chiruko-stand-running.png`: 透過済みミニキャラ立ち絵（移動）
- `assets/chiruko-stand-sleeping.png`: 透過済みミニキャラ立ち絵（おやすみ）
- `assets/chiruko-stand-sitting.png`: 透過済みミニキャラ立ち絵（待機）
- `booth-copy.md`: BOOTHや紹介欄に貼りやすい文章案

## 使い方

1. `index.html` をブラウザで開きます。
2. GitHub Pagesで公開する場合は、この `outputs` 内のファイルをリポジトリの公開対象に配置します。
3. 画像を変更したい場合は、`assets` 内の画像を同じファイル名で差し替えます。

## デザイン方針

- 余白を広めに取った小さなLP
- パステルピンク、ミント、レモン、ラベンダーのやわらかい配色
- 漢字対応の `M PLUS Rounded 1c` を優先した丸ゴシック系フォント指定
- 黒金デザインそのものには寄せず、ちる子の黒い衣装が映える明るい背景
- ナビゲーションや仮の公開情報を省いた、本番向けの紹介ページ
