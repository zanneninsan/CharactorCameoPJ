# デスクトップちる子 静的サイト

`/zannenin/desktopchillko/` として公開する、残念院ちる子のデスクトップアクセサリ紹介ページです。

## ファイル

- `index.html`: 公開ページ本体
- `styles.css`: ページ専用スタイル
- `assets/`: 紹介画像とミニキャラ立ち絵

## デザイン方針

- 本番用の小さな紹介ページとして、長いLPではなくシンプルな構成にする
- 余白、パステルカラー、丸み、キュートなフォントを主役にする
- Character Cameo PJ本体のデフォルトデザインには寄せない
- ナビゲーションや仮の配布情報は置かない
